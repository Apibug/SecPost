# 飞鱼星 家用智能路由 cookie.cgi 权限绕过

## 漏洞描述

飞鱼星 家用智能路由存在权限绕过，通过Drop特定的请求包访问未授权的管理员页面

## 漏洞影响

```
飞鱼星 家用智能路由
飞鱼星 企业级智能上网行为管理系统
```

## 网络测绘

```
title="飞鱼星家用智能路由"
```
