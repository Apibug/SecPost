# 金和OA C6 OpenFile.aspx 后台越权敏感文件遍历漏洞

## 漏洞描述

金和OA C6 存在后台越权敏感文件遍历漏洞，普通用户通过遍历特殊参数可以获取其他用户上传的敏感文件

## 漏洞影响

```
金和OA C6
```

## 网络测绘

```
app="Jinher-OA"
```
