# 智慧校园管理系统 前台任意文件上传漏洞

## 漏洞描述

智慧校园管理系统前台注册页面存在文件上传，由于没有对上传的文件进行审查导致可上传恶意文件控制服务器

## 漏洞影响

```
智慧校园管理系统
```

## 网络测绘

```
body="DC_Login/QYSignUp"
```
