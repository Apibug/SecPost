# 安天 高级可持续威胁安全检测系统 越权访问漏洞

## 漏洞描述

安天 高级可持续威胁安全检测系统 存在越权访问漏洞，攻击者可以通过工具修改特定的返回包导致越权后台查看敏感信息

## 漏洞影响

```
安天 高级可持续威胁安全检测系统
```

## 网络测绘

```
title="高级可持续威胁安全检测系统"
```

登录页面如下



![](./images/202202091835601.png)

![](./images/202202091835128.png)

![](./images/202202091836868.png)
