# 新开普 前置服务管理平台 service.action 远程命令执行漏洞

## 漏洞描述

新开普 前置服务管理平台 service.action 接口存在远程命令执行漏洞，攻击者通过漏洞可以获取服务器权限

## 漏洞影响

新开普 前置服务管理平台

## 网络测绘

```
title="掌上校园服务管理平台"
```
