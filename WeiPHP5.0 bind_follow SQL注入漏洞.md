# WeiPHP5.0 bind_follow SQL注入漏洞

## 漏洞描述

Weiphp5.0 所有使用了 wp_where() 函数并且参数可控的SQL查询均受到影响，前台后台均存在注入。

## 漏洞影响

```
Weiphp5.0
```

## 网络测绘

```
app="WeiPHP"
```
