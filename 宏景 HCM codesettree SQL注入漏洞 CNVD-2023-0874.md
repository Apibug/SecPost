# 宏景 HCM codesettree SQL注入漏洞 CNVD-2023-0874

## 漏洞描述

宏景 HCM codesettree 接口存在SQL注入漏洞，攻击者通过漏洞可以获取到登陆系统的账号密码和数据库信息

## 漏洞影响

宏景 HCM

## 网络测绘

```
app="HJSOFT-HCM"
```

## 漏洞复现

登陆页面
