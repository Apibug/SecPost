# 红帆OA ioFileExport.aspx 任意文件读取漏洞

## 漏洞描述

红帆OA ioFileExport.aspx文件存在任意文件读取漏洞，攻击者通过漏洞可以获取服务器敏感信息

## 漏洞影响

```
红帆OA
```

## 网络测绘

```
app="红帆-ioffice"
```
