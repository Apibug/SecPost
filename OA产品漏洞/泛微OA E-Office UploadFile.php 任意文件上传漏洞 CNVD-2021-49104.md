# 泛微OA E-Office UploadFile.php 任意文件上传漏洞 CNVD-2021-49104

## 漏洞描述

在/general/index/UploadFile.php中上传文件过滤不严格导致允许无限制地上传文件，攻击者可以通过该漏洞直接获取网站权限

## 漏洞影响

```
泛微OA V8
```

## 网络测绘

```
app="泛微-EOffice"
```
