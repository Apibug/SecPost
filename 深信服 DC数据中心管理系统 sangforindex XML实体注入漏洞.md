# 深信服 DC数据中心管理系统 sangforindex XML实体注入漏洞

## 漏洞描述

深信服 DC数据中心管理系统 sangforindex 接口存在XML实体注入漏洞，攻击者可以发送特定的请求包造成XML实体注入

## 漏洞影响

深信服 DC数据中心管理系统

## 网络测绘

```
"SANGFOR 数据中心"
```
