# 飞视美 视频会议系统 Struts2 远程命令执行漏洞

## 漏洞描述

飞视美 视频会议系统 Struts2组件存在远程命令执行漏洞，通过漏洞攻击者可执行任意命令获取服务器权限

## 漏洞影响

```
飞视美 视频会议系统
```

## 网络测绘

```
app="飞视美-视频会议系统"
```
