# 中科网威 下一代防火墙控制系统 download.php 任意文件读取漏洞

## 漏洞描述

中科网威 下一代防火墙控制系统 download.php 任意文件读取漏洞, 攻击者通过漏洞可以读取服务器上的文件

## 漏洞影响

```
中科网威 下一代防火墙控制系统
```

## 网络测绘

```
body="Get_Verify_Info(hex_md5(user_string)."
```
